INSTRUCCIONES PARA CORRER "LoveITLA"

1. Abre una terminal.
2. Ejecuta: npm install
3. Luego: npm start
4. Visita http://localhost:3000 en tu navegador.

Este sistema te permite:
- Crear cuenta
- Iniciar sesión
- Hacer confesiones de amor (públicas o privadas)
- Leer confesiones públicas (sin iniciar sesión)
- Leer privadas (sólo si inicias sesión)
